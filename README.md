# SportsPage
运动页，一个基于社交的体育类APP，具体相关内容可以去我的博客查看。

# 项目结构

* android 	*安卓项目源代码*
* server 	*后台服务器代码*
	* webapi
	* tools
* doc		*项目开发过程中的一些文档*
* script	*脚本*
* db		*数据文件*
* screenshot *项目截图【UI上可能有些不一样，但是功能是差不多的】*

# 项目说明
项目中使用了第三方的组件，如《融云IM》，《微信SDK》，《ping++》，《百度地图》等，具体这些的框架代码就不提供了。
